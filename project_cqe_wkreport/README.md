# CQE Weekly Report System

This folder is the working area for the new CQE weekly report system.

## Current status

- `V7/` keeps the previous prototype version.
- `docs/v7-analysis.md` records what was recovered from V7.
- `docs/requirements.md` defines the first-pass scope for the new system.
- `index.html` is a standalone browser entry that works without Node.js.

## Direction

The new system should move from a single-page report generator to a maintainable web system with:

- structured weekly report records
- issue / interaction / activity data modules
- reusable project mapping and terminology dictionary
- exportable weekly summaries
- room for review flow and permissions later
