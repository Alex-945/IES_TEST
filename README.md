# IES_TEST Workspace

`IES_TEST` is a workspace repository that groups documents and multiple independent projects. The root should only keep workspace-level entry files and shared structure.

## Layout

```text
IES_TEST/
  docs/
  projects/
  .gitignore
  README.md
```

## Folders

- `docs/`: shared documentation and reference material for the workspace.
- `projects/`: independent project folders grouped by domain or lifecycle.

## Project Groups

- `projects/cqe/`: CQE-related tools and prototypes.
- `projects/active/`: currently active projects.
- `projects/pending/`: projects that are being organized, evaluated, or prepared.
- `projects/archive/`: archived projects kept for reference.

## Entry Points

- Project index: `projects/README.md`
- Development notes: `docs/開發清單.md`
- Design references: `docs/design_principles/`
